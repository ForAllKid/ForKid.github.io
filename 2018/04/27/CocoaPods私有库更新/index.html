CocoaPods私有库更新

Apr 27, 2018 | 技术

1.修改.podspec文件中的版本号；

s.version = xxxx //如 "1.0.1" 修改1.0.0版本，不能是之前的版本，要不提交不成功

2.上传到远程仓库；

$ git add . ;  

$ git commit -m “XX”  ; 

$ git push origin yourbranch。

3.打tag并上传；

$ git tag 'xxxx' ;//和s.version版本号一致

$ git push --tags;

4.验证pod并提交到pod。

$ pod spec lint --allow-warnings    //远程本地pod验证

$ pod repo push repo yourpods.podspec   //提交pod到自己的私有索引库，remoteLib是之前生成的本地    repo名

$ pod search RemoteLibQiuyh   //查询库
